q1.py is name of the file to be executed.
It asks for character and dispalys stocks starting from it and also writes it into file.
It filters the data based on change percentage.

All output file contains data based on filtered input data.
It generates two output files:
    1. avg_output.txt:
        It contains three rows:
            open (open average)
            low  (low average)
            high (high average)
    
    2. stock_output.txt:
        It contains information of stocks whose first letter matches with given input.
        Information about each stock is space separated and Information of every stock is separated by new line.
